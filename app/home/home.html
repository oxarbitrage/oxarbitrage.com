Oxarbitrage builds software for the blockchain community in general, the graphene blockchains specifically and more granularly: The Financial Decentralized exchanged named Bitshares.

For fair, decentralized, faster and better open markets, oxarbitrage innovates in the blockchain area related to finance.

Projects:

The most active projects that oxarbitrage is working currently are the followings

Bitshares-Core
The C++ bitshares-core is where most of the work of oxarbitrage is currently done.
Bitshares Open Explorer
Angular explorer for bitshares that has became one of the most used.
Bitshares UDF wrapper
Wrapper to integrate Bitshares market data with Traving View charts.
Bitshares ES wrapper
Easy wrapper to make use of the elasticsearch bitshares-core plugin for full account data storage.
Bitshares Account Exporter
Download full account history from the bitshares blockchain.
Bitshares Python Backend
Python FLASK api allow other programs to connect via web easy.
